# Glossary

Human-readable mirror of [content/glossary.yaml](../content/glossary.yaml). Update both when adding or changing terms.

| Term | Description |
| --- | --- |
| **RAG** | Retrieval-Augmented Generation; combines source retrieval with language-model generation so answers stay grounded in real documents. |
| **Hallucination** | A model answer that sounds plausible but is factually wrong or unsupported by any source. |
| **Human-in-the-loop** | A workflow where a human reviews or approves outputs that have legal, financial, or social impact. |
| **Digital sovereignty** | Public control over data, models, and infrastructure in line with public values. |
| **Doenvermogen** | The Dutch term for citizens' practical ability to navigate administrative processes, especially under stress. |
| **LLMOps** | Operational practices for deploying, monitoring, and maintaining large language models in production. |
| **Guardrail** | A rule, filter, or policy that constrains assistant behaviour to acceptable inputs and outputs. |
